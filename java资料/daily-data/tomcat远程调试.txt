一：服务器端打开tomcat的远程调试功能
	1.配置远程服务器的jvm启动参数
	－XDebug（打开调试） -Xrunjdwp:transport=dt_socket,server=y,suspend=n(or y ps:y代表客户端连接上才启动jpda－JAVA Platform Debug Architecture),address=8788

	2.或者以catalina jpda start方式启动 在tomcat/bin/startup.sh 中修改 start ＝> jpda start  (最末行exec "$PRGDIR"/"$EXECUTABLE" start "$@")

	一般会复制一份catalina.sh，重命名为jpda.sh，当需要远程调试时，启动jpda.sh即可，默认端口号是8000

二：客户端连接服务器
1.Debug Configurations－>Remote Java Application -> new ->Host:远程主机 Port：jpda设置的端口号
2.在对应的入口打上断点即可调试
